Title: Gateway 配对故障排查
Date: 2026-02-22
Category: 技术

今天遇到了 Gateway 配对故障，记录一下排查过程。

## 症状

```
gateway closed (1008): pairing required
```

## 根因

CLI 请求更高的 scopes (`operator.pairing`, `operator.approvals`)，触发 scope-upgrade。

## 解法

1. 打开 Dashboard
2. 点击 Approve 设备配对请求
3. 重新连接

## 教训

- 先诊断（看日志的 reason 字段）再行动
- 不要乱删文件
- 记录到 `RUNBOOK.md` 第10条


排查问题时，最重要的不是「怎么做」，而是「先看什么」。
